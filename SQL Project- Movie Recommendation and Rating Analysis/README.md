# Movie Recommendation & Rating Analysis System

## Overview

This project simulates a basic movie recommendation system using SQL. It analyzes user behavior, ratings, and watch history to generate insights and recommendations.

## 🛠 Technologies Used

* MySQL
* SQL (Joins, Aggregations, Indexing)

## Features

* Top-rated movie analysis
* Genre-based insights
* Most active users tracking
* Trending movie detection

##  Key Concepts Used

* Joins (INNER JOIN)
* GROUP BY, ORDER BY
* Aggregate functions (AVG, COUNT)
* Foreign Keys

## 🚀 Outcomes

* Built a relational database system
* Implemented real-world recommendation logic using uniform data
* Generated meaningful insights from data
